# Dental Website Template

A clean, responsive single-page website template for dental clinics. Built with HTML, CSS, and vanilla JavaScript—no frameworks required.

## Features

- **Responsive layout** — Works on desktop, tablet, and mobile
- **Sections** — Hero, Services, About, Team, Contact
- **Accessible** — Semantic HTML and ARIA where needed
- **Modern styling** — Custom CSS with DM Sans & Playfair Display fonts
- **Mobile menu** — Hamburger navigation for small screens
- **Call-to-actions** — Book appointment and contact links

## Project Structure

```
dental-website-template/
├── index.html      # Main page
├── css/
│   └── styles.css  # Styles
├── js/
│   └── script.js   # Scripts (e.g. mobile menu)
└── README.md
```

## Getting Started

1. **Clone or download** this repository.
2. **Open** `index.html` in a browser, or serve the folder with any static server.

No build step or dependencies required.

## Customization

- **Clinic name & branding** — Update the logo text and page title in `index.html`.
- **Content** — Edit section copy, services, team names, and contact details in `index.html`.
- **Colors & fonts** — Adjust variables and styles in `css/styles.css`.
